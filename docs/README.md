## Quick Reference

| Asset Type | Dimensions | Format |
|------------|------------|--------|
| Character sprites | 16×24 px | PNG-24 |
| Tiles | 16×16 px | PNG-24 |
| Portraits | 128×192 px | PNG-24 |
| UI Icons | 16×16 px | PNG-24 |

### Color Palette

```
BLACK:  #000000 #0A0A0A #1A1A1A #2A2A2A #333333
GRAY:   #666666 #888888 #B0B0B0 #D4D4D4
WHITE:  #EDEDED #FFFFFF
GOLD:   #D4AF37 #B8860B (sacred objects only)
GREEN:  #4AF2A1 (UI only)
```

## The Vision

This game's visual identity is punk iconography. **Death to the World** meets Byzantine mosaics. Stark, beautiful, confrontational.

Not a "Christian video game" — an Orthodox experience that stands on its own as art.

---

## Getting Started

1. Read ART_STYLE_GUIDE.md thoroughly
2. Install Aseprite (or LibreSprite)
3. Set up the color palette
4. Create a test 16×16 tile
5. Create a test 16×24 character
6. Export, test in-game
7. Pick an asset from ASSET_LIST.md
8. Make it real
